# Задание 2
Необходимо суммировать числа, записанные в перевернутом виде и вывести их сумму также в перевернутом виде.

## Сборка
```
gradle build
```

## Запуск

Переместиться в build/classes/main (появится после успешной сборки)
```
cd bmstu-hackathon-2017/firststep/ex2/build/classes/main
```

Ввод:
```
echo "3 24 1 4358 754 305 794" | java com.github.aminbenarieb.bmstuhackathon2017.firststep.ex2.Application
```

Вывод:
```
34
1998
1
```
